# Repositorio del sitio web del Restaurante Rosita

Este repositorio contiene la informacion necesaria para bajarse y modificar el sitio web "Restaurante Rosita".

Si quieren colaborar en este proyecto, lean la seccion Colaborar.

## Instalar este sitio web localmente

Para instalar el sitio web, sigan estos pasos:

1. Hacer `git clone` del proyecto
2. Hacer `git commit` de sus cambios
3. Hacer `git push` al repositorio remoto
